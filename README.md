# Project Socrates: Socratic AI Assistant

A full-stack AI application designed to engage users in philosophical dialogue. This project was completed as part of the Valearnis Internship task.

## Live Demo
- **Frontend:** https://ssocrates-ai-two.vercel.app
- **Backend/API:** https://huggingface.co/spaces/sunilbastola/socrates

## 🛠️ Technical Stack
- **Frontend:** React.js, Tailwind CSS, Vite
- **Backend:** FastAPI (Python), Docker
- **NLP/AI:** spaCy (Natural Language Processing), Hugging Face Inference API
- **AI Moedl Used:** Meta Llama 3.1 (8B Instruct)
- **Deployment:** Vercel (Frontend), Hugging Face Spaces (Backend)

## Key Features
- **Socratic Inquiry:** Custom-tuned logic to respond with philosophical depth.
- **NLP Analysis:** Uses spaCy to perform lemma extraction on user inputs to understand core concepts.
- **Decoupled Architecture:** Clean separation between the AI service and the UI for scalability.
